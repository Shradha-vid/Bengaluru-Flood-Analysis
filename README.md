# Bengaluru-Flood-Analysis
# Bangalore Flood Prediction & Risk Categorization System
Overview
This project focuses on predicting flood-prone regions in Bangalore using geospatial and SAR data, and classifying areas into risk categories (High, Medium, Low) using machine learning.

Problem Statement
Flooding in urban areas like Bangalore causes significant damage. Identifying high-risk zones in advance can help in disaster management and planning.

Approach
* Collected and processed geospatial and SAR data
* Performed data preprocessing and feature extraction
* Applied machine learning models for flood prediction
* Classified regions into risk categories

Tech Stack
* Python
* Pandas, NumPy
* Scikit-learn
* GIS / SAR Data Processing

Results
* Successfully classified flood-prone regions into risk categories
* Generated visual outputs for better understanding of affected areas
